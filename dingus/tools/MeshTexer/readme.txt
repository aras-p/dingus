MeshTexer is a tool to generate seamless textures onto arbitrary 3D models.
Given a mesh with unique UV mapping and a tileable 2D texture, it generates
a texture that nicely "wraps around" this 3D model.

MeshTexer is part of 'dingus' 3D engine and tools, http://dingus.berlios.de

The complete documentation is online:
  http://dingus.berlios.de/index.php?n=Main.MeshTexer
